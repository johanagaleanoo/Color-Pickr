Color Pickr - Proyecto Personal

Color Pickr es una aplicación web que permite al usuario subir una imagen, seleccionar cualquier color dentro de la misma y copiar su valor en formato RGB o HEX para poder utilizarlo en algún proyecto.

Funcionalidades:
- Carga de imágenes
- Seleccionar un color
- Convertir a RGB y HEX
- Copiar los valores 

Tecnologías utilizadas:
- Javascript
- HTML5
- CSS
- Bootstrap
